HCI Principles Applied to This Site

Every design decision on this website is grounded in HCI theory from ARIN5301. Below are the key principles and how they are applied.

Usability Heuristics

  • Consistency Internal, external, and metaphorical consistency across the interface.
  • Mapping Match between system and real world — use language users understand.
  • Minimalism Every extra element competes with relevant information.
  • Freedom User control — undo, redo, emergency exits.
  • Flexibility Accelerators for experts, support for novices.
  • Recognition Make options visible rather than requiring recall.
  • Visibility Keep users informed about system status.
  • Error Prevention Prevent errors before they occur through constraints.
  • Error Recovery Help users recognize, diagnose, and recover from errors.
  • Help Provide help and documentation when needed.

Gestalt Principles

  • Proximity Close elements are perceived as related groups.
  • Similarity Similar elements are perceived as belonging together.
  • Continuation Elements on a line or curve are seen as related.
  • Closure The mind completes incomplete shapes.
  • Figure-Ground Users distinguish foreground from background.
  • Common Fate Elements moving together are grouped.
  • Symmetry Balanced elements are perceived as unified.

Design Levels

  • Visceral Immediate emotional reaction — sensation, first impression.
  • Behavioral Usability and function — does it work well?
  • Reflective Meaning and self-image — what does it say about me?
Design Notes

How this site practices HCI

🎨

Color as Identity

Cultural palette connecting ink painting traditions to visual design. Celadon accent distinguishes course content (H1 Consistency).

✏️

Typography as Hierarchy

Three typefaces for three roles: serif display, sans body, mono metadata. Supports Recognition over Recall (H6).

◻️

Whitespace as Design

96-128px section spacing creates visual "rooms" using Gestalt Proximity. 720px content width for optimal reading.

✨

Motion as Feedback

GSAP scroll animations provide Visibility of System Status (H7). All animations respect prefers-reduced-motion (H5).
